# COMMAND LINE MLB BOX SCORES#

### What is this for? ###

* Hate having to search for MLB scores in the browser? Of course you do. 
* Version 1.0

### How do I get set up? ###

Fork this repo and add your own access token. 

### Get the box score
``cd mlb-box-scores``

``python mlb.py -d 20140520 -t cardinals``

### See optional arguments
``python mlb.py -h``
